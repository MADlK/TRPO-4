команда 'cd'-позволяет перейти и начать работу по введенному пути
команда 'git init'-инициализация локального репозитория
команда 'echo'-создание файла
команда 'git add'-добавление файла в репозиторий
команда 'git commit'- создает комментарий для добавленого в репозиторий файла
команда 'git branch/git switch/git merge'-создание, переключение и слияние веток репозитория
команда 'git status'-выводит текущий статус репозитория
команда 'git remote/git push/git pull'-позволяет взаимодействовать с удаленныым репозиторием
